Le déclencheur ne s'activera que pour les tâches dont l'expression régulière valide le champ "référence". Aucun filtrage ne sera appliqué si le champ est laissé vide.

Les éventuels groupes de correspondance de l'expression régulière seront exportés dans les variables d'environnement suivantes : KANBOARD_TASKREF_{index du groupe}

Par exemple :

L'expression régulière ([A-Z]+)-([0-9]+) validera la référence ABC-12345 et produira les variables d'environnement suivantes :

  • KANBOARD_TASKREF=ABC-12345
  • KANBOARD_TASKREF_0=ABC
  • KANBOARD_TASKREF_1=12345
